# Branded QR Code Generator

Create scannable QR codes with custom colours and lightweight branding. Preview updates live as you adjust settings, and download a PNG in one click.

Live tool: `https://urlcv.com/tools/branded-qr-code-generator`

## Features

- URL validation (requires `http(s)://` and a dot)
- Preset themes: Light, Dark, Brand Blue, Monochrome
- Foreground/background colour pickers
- Size + quiet zone (minimum enforced to protect scanability)
- Optional label (above / below / none)
- Live scanability score (Excellent / Good / Risky) based on contrast + size
- PNG download with label baked into the image

## Notes

This tool runs entirely in the browser (Alpine.js + QRCode.js). No URL is uploaded to the server.
